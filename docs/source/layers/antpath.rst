Interactions
------------

Like most widgets in ipyleaflet, the ``AntPath`` can be dynamically updated from Python.

.. code::

    # Update the color
    ant_path.color = 'red'

    # Update the path
    ant_path.locations = [[51.185, 6.773], [51.326, 6.726], [51.497, 6.714]]
